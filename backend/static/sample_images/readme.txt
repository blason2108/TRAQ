# Sample Images Folder

Place sample images here that you want to display when inspecting violations.

### How to name your images:
1. **Case-Specific Image:** Name the image after the Case ID (e.g., `VIOL-DEMO-1782225299.jpg`). This image will show only for that specific case.
2. **Type-Specific Image:** Name the image after the Violation Type (e.g., `No Helmet.jpg` or `Running Red Light.jpg`). This will act as the default image for all cases of that violation type.

*Ensure files are saved with the `.jpg` extension (lowercase).*
